# Budget Tracker

A personal budget and expense tracking app.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Recharts

## Getting Started

```sh
npm install
npm run dev
```

Runs at http://localhost:8080.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm test` | Run tests |
